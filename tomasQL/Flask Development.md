### Flask micro-framework

**Installing VENV**
  
    python3 -m pip install --user virtualenv  
  
**Setting up virutal enviroment**
  
    python3 -m venv <folder name to the venv>
    *add venv in file .gitignore
      
    source env/bin/activate  
    deactivate  
  
**Installing Flask**
  
    pip install Flask
  
**Run Flask app**
  
    flask -app <nameapp> run
